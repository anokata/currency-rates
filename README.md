Currency rates
==============
Web application that display currency rates. 

Load data from http://fixer.io/

Using flask as framework. 

Display currency rates data with possibility to query in natural language(russian).

Requirements:
* flask
* requests
* pymorphy2-dicts-ru
* pymorphy2
* pytest (if need testing)

Running:

    make run

Testing:

    make test
